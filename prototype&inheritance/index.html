JS Coding Assignment-Prototype and Inheritance

    Question1-Create 2 objects - parent and child. Attach methods to parent and use those methods in child object 
    using parents prototype

    let parent={
        greet : function(){
            console.log("Good Evening");
        }
    }
    let child=Object.create(parent);
    console.log(child);
    child.greet();

    explanation-
    code creates an object parent with a method greet, and an object child created using the Object.create method that inherits from parent.
    When console.log(child) is called, it will output an empty object, as child has no properties of its own. However, it inherits the greet 
    method from its prototype parent.
    When child.greet() is called, it will invoke the greet method from its prototype parent, which will log "Good Evening" to the console.

    Question2-Write code to explain prototype chaining
    function print(num){
        return new Promise((resolve)=>{
            setTimeout(()=>{
                 console.log(num);
                 resolve()
             },1000)
         })
      }
      print(1) 
      .then(()=>print(2))
      .then(()=>print(3))
      .then(()=>print(4))
      .then(()=>print(5))
      .then(()=>print(6))
      .then(()=>print(7))

    explanation-
    here multiple .then methods are used which indicates prototype chaining. The code calls print(1) and 
    chains a series of then() methods that each call print() with a new number once the previous Promise resolves.
    This creates a sequence of delayed console logs that will execute in order.
