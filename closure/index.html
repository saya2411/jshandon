question2- 
        let count = 0;
        (function () {
            if (count === 0) {
                let count = 1;
                console.log(count); // What is logged?
            }
        console.log(count); // What is logged?
        })();
        

            Output is-
            1
            0
        
        because when control assigns the 0 to count variable(global variable), function() will be invoked which is an IIFE(Immediately Invoked Function Execution), 
        so first control will check whether count value is equal to 0 or not, here it is true, so it will go inside the if block, there count is reassingned 
        so the reassingned value of count will be considered and the control will print the reassigned value that is 1. Now the control will come out of the 
        if block, then again there is a print statement,so this stmt will print count value as 0, since count is 0(from global scope)

question4-
        Write a code to calculate area of a rectangle using inner function. In this case outer function should accept 
        parameter length and inner function should accept parameter breadth.

            function length(length) {
                function breadth(breadth) {
                  return length * breadth;
                }
                return breadth;
              }

              const areaOfRectangle = length(20);
              console.log("Area of Rectangle is: "+areaOfRectangle(10));
        
            output--
            Area of Rectangle is: 200
        

            here length() is having a length as a parameter(outer function), which returns breadth as a result of inner function breadth having breadth as a parameter,
            multiples length and breadth. So, when length function is called with 20 as an argument, the value gets stored in the areaOfRectangle varible, which then 
            passess 10 as an argument, which is taken by the inside function breadth, which then multiples length and breadth, resulting breadth to the outer function
            which then prints the value 200 as the areaOfRectangle.
